### Installing

to have the project
```
$git clone https://github.com/clotair/gps_tracker.git
$cd gps_tracker
$npm install
```
### Start project

to launch the project run the following commande
```
$npm start
```
#### Or
```
$node server.js
```
the project is served on `localhost:3000/`
